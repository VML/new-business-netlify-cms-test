Flexbox Grid

  • 12 columns
  • Use col- plus number of columns to define width. Example: col-4
  • Add responsive modifiers to trigger different column widths, sorting, or positioning by breakpoint xs:, sm:, md:, lg:, xl:
  • A set of columns must be wrapped with the .row wrapper.
  • Any set of columns inside of a .row that do not equal to 100% will wrap to a new line.

Auto

  • Add any number of columns to a row and each column will be the same width.
  • Perfect for when having an odd numnber of columns that is not divisible by 12.
  • Can be utilized to fill up remaining amount of space in conjunction with a column that has a defined width.

Variable

  • Takes on the width of the content inside of the column.
  • Best utilized with content that has a max-width or set width.
  • Variable columns work great with positioning helper classes.
  • If the column does not fit inside of the row, the column will wrap to new line.
