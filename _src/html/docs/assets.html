Assets (update the content on this page)

  • Individual spotlight blocks to feature content such as agency work or case studies.
  • The spotlight scales in accordance to the size of picture supplied.
  • Combine with the Grid to create the desired layout.

Parameters

Key Value Default
title Sets the title below picture. --
iconId Use iconography to place an icon over the picture when hovered. --
url Sets the url for the clickable link or video embed url. --
isVideo Set true for video or false for links. false
picture See picture component for parameters --
