Type Sets

  • Instead of being able to apply multiple classes to control the font-size directly. As an author you are instead choosing from pre-defined type-sets. These type-sets when combined with a modular design add consistency to your project.
  • Some of the type-sets will have built in line-height, letter-spacing, transforms, or responsive sizing. Now if we want an h3 element to look like an h1, we can simply do so by applying .type-h1 to the h3 element.
  • You can further modify the styles of your typography by combining font-weight, letter-spacing, line-height, style & decoration, and color.
