Map

  • Add a map with markers and the map will center over the markers area.
  • Images are in the images/maps directory and modals are in _partials.
  • Use the Twig {% verbatim %} {% include{% endverbatim %} to show the map component, components/map.html, and modal component components/modal.html.
  • GoogleMapsLoader.KEY in maps.js is domain specific to localhost and vmlyrconnect.com. This requires billing and is on Besim's personal account, so low traffic sites only on this key.

Parameters

Key Value Default
mapSrc location is the name of the marker location title.
id is a unique value, identical to the modals id.
lat: latitude of the location.
lng: longitude of the location.
icon: star, bar, coffee, shopping, hotel, building, memorial, hash-tag, vmlyr		 --
id This is a unique value for the marker, identical to the map id. --
imgFileName The path to the image with the filename for the background image at the top of the modal. --
locName The title text of the location over the background image. --
excerpt This is an optional area to add text in the first section. --
details Each parameter is optional.
title is above the text.
text is the visible text
url creates a link for email, phone, map for directions, or a website
icon: directions for car icon, phone for phone icon		 --
