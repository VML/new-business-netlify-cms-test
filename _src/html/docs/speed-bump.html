Speed Bump

  • Uses the grid layout to create divided sections but with ul and li.
  • Use classes to adjust font color, font size, background color, etc.
  • Use class speedBump__col--light or speedBump__col--dark on the li element for 2 additional background colors.
  • To adjust the number of boxes, add or remove the number of li tags. Be sure to adjust your column classes.
  • Update the p text, link href url to the download file or landing page, link text, icon parameters, and class for button color.
  • See iconography for working with icon and buttons for working with button.
