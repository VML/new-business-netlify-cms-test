Spotlight

  • Individual spotlight blocks to feature content such as agency work or case studies.
  • The spotlight scales in accordance to the size of picture supplied.
  • Combine with the Grid to create the desired layout.

Parameters

Key Value Default
title Sets the title below picture with a primary and an optional sub.
overlay - boolean to move the primary text over the image with an optional icon.
overlayIcon - boolean to display the icon below the title.
overlayHoverIcon - boolean to display the icon below the title on hover.
overlayText - title text over the image.
overlayTextHover - title text over the image on hover. 		--
hotspot Add hover features by setting newWindow, url and icon, isVideo, or disable features. disabled: false
extras Add links below the image by setting the url, title, newWindow, video, and version optional features. --
picture See picture component for parameters --
