Figure

  • Use the Twig {% verbatim %}{% include{% endverbatim %} to show the figure component, components/figure.html.
  • Rely on Contentful media assets or self hosted image.
  • Combine with grid and other layout utility classes to create ideal design.

Where do I find the Contentful asset id?

  • Login to contentful.com.
  • Navigate to "Media" located in the header navigation.
  • Search or look through the list of "Published" assets. Once desired asset is selected, click it for additional details.
  • On the asset detail page, find the "Info" button on the top right corner of the page to retrieve the asset ID.

Parameters

Key Description Contentful only?

srcPath Instead of using a Contentful asset, rely on self hosted or CDN source. No
id Contentful media asset id. Yes

width Set picture width. If left empty, will default to image source width. Yes
placeholderSize Define placeholder dimension for lazy loading. Default: 10 10. The values should be unitless.
